# PHP Google Translate Clean Switcher

Clean Google Translate language switcher for PHP websites without the ugly Google top bar.

## Features

- Clean modern language selector
- No Google top banner
- Cookie based language memory
- Works on any PHP website
- Lightweight single file

## Installation

1. Download `lang-switcher.php`
2. Place it in your project
3. Include it right after `<body>`

```php
<?php include 'lang-switcher.php'; ?>
